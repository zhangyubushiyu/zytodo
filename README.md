# 待办管理系统

一个功能完整的团队待办管理系统，支持多设备数据同步。

## ✨ 功能特性

- 📅 **日历视图** - 直观的日历界面，快速查看和管理待办
- 👥 **团队协作** - 支持多负责人，方便团队协作
- 🏷️ **分类管理** - 自定义分类，颜色标识，清晰分类
- ⏰ **提醒功能** - 支持待办提醒，不错过重要事项
- 📱 **跨设备同步** - 基于Bmob云数据库，多设备数据实时同步
- 🔄 **数据导入导出** - 支持JSON格式导入导出，数据备份无忧
- 📋 **继承功能** - 一键继承昨天的待办，智能去重

## 🚀 在线访问

访问地址：https://你的用户名.gitee.io/zytodo/

## 📦 技术栈

- **前端**：原生HTML5 + CSS3 + JavaScript
- **数据库**：Bmob云数据库
- **部署**：Gitee Pages

## 🛠️ 本地运行

```bash
# 克隆仓库
git clone https://gitee.com/你的用户名/zytodo.git

# 进入项目目录
cd zytodo

# 启动本地服务器
python3 -m http.server 8080

# 访问 http://localhost:8080
```

## 📖 使用说明

### 1. 添加分类
- 点击左侧"分类管理"
- 输入分类名称，选择颜色
- 点击"添加"按钮

### 2. 添加负责人
- 点击左侧"负责人管理"
- 输入负责人姓名和职位
- 点击"添加"按钮

### 3. 添加待办
- 点击右侧"快速添加待办"
- 填写待办内容、分类、负责人等信息
- 点击"添加"按钮

### 4. 继承昨天的待办
- 选择今天的日期
- 点击"继承昨天"按钮
- 系统会自动跳过重复的待办

### 5. 数据导入导出
- 点击右上角"导出数据"按钮，导出JSON文件
- 点击"导入数据"按钮，选择JSON文件导入

## 🔧 配置说明

### Bmob配置

1. 注册Bmob账号：https://www.bmob.cn/
2. 创建应用
3. 创建数据表：Todo、Category、Responsible
4. 修改 `js/bmob.js` 中的配置：

```javascript
const BmobConfig = {
    secretKey: '你的Secret Key',
    apiSecurityCode: '你的API安全码'
};
```
